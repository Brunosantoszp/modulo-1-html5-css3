Adicionando favicon:

Nessa página vamos colocar o favicon usando a tag <Link em seguida escolhendo favicon>, essa tag obrigatoriamente deve estar dentro da tag <head> de preferência acima da tag <title>
Considere ter um arquivo icon na mesma pasta do documento index.html;

Algumas considerações antes de escolher o favicon é o formato, o PNG e o ICON sendo icon o mais indicado, veja onde encontra;

Para escolher um icon aleatório em bibliotecas o softwere é:

  • IconArchive;

Para desenhar o favicon o softwere é:

  • Favicon.cc;

Para converter texto, emoji e imagem em formato png ou icon o softwere é:

  • Favicon.io;

Comando depois da tag <Link:favicon> dentro das aspas;
Ctrl+Space;

Adicionando imagem:

Algumas considerações antes de colocar imagens são:

  • Formato;
  • Tamanho;
  • Direitos autorais;
Todas essas considerações pode impactar significamente seu site tornando-o mais acessível e leve ou menos acessível e pesado e principalmente atenção total com imagens que tem direitos autorais;

Há sites que contém imagens e vídeos livres de direitos autorais, vamos nos aprofundar mais nisso quando for colocar vídeos na página
Agora para editar arquivos JPEG,PNG e JPG o progama indicado além de ser um softwere gratuito é o Gimp:

  • PNG quando se quiser trabalhar com imagens de fundo transparente;
  • JPEG e JPG quando se quiser trabalhar com imagens melhores compactadas e com a qualidade melhorada;

página, ficou com uma aparência de zoom;

Adicionando áudio:

Algumas considerações a serem feitas é a configuração de como será exibido o áudio e o tipo do áudio.
É ideal que tenha o áudio disponível em todos os formatos por isso o ideal é aicionar a tag que permite colocar todos os fomatos que são:

  • Formato MP3;
  • Formato OGG;
  • Formato WAV;
Sendo o formato wav o mais pesado e pouco recomendado por conta disso.

A primeira forma para inicio é cococar uma música só:

Fomato mp3/mpeg;

O segundo comando é o mais indicado para um site público pois permite adicionar diferentes formatos para cada tipo de navegadores e dispositivos, a tag audio para adicionar vários áudios precisará combinar a tag <source src=" ">


Adicionando vídeo hospedado e incorporado:

A desvantagem de ter vídeos hospedado no próprio site é que consome muito tráfego de banda;
É mais vantajoso ter vídeos que seja incorporado de outros sites ou plataforma por exemplo o YouTube e Vimeo;

Se escolher vídeo hospedado deve ficar atento no tamanho, formatos e direitos autorais, para isso usaremos uma ferramenta para manipular o tamanho e as dimensões dos vídeos e a ferramenta é HandBrake.

Considerações extras:

Parâmetros do áudio:
O comando controls autoplay loop na linha da tag <audio> faz o áudio reproduzir automaticamente assim que carregar o site;

O comando preload: Auto,Metadata e None;
O auto considera o site carregado só depois que o áudio todo carregar;
O metadata mostra as configurções de reprodução de áudio;
O none não mostra nenhuma configuração apenas o botão play do áudio;

Parâmetros do vídeo:
O comando loop na linha da tag <video> recomeça sempre que terminar o vídeo;
O comando poster permite adicionar thumbnail no vídeo;

Nas linhas das tags de mídia visual como vídeos e imagens os comando width e height é essencial caso o arquivo esteja tomando a página inteira para redimensionar visualmente o arquivo dentro da página, este é altura e aquele é largura;
