Estilo exteno, downloads e imagens dinâmica

Aplicando estilo externo:

Com estilo externo é possível modificar o estilo de todas as páginas;
Existe três tipo de estilo, o estilo linha, o interno e o externo mas linha sobrepôe as duas e o estilo mais indicado para muitas páginas é o estilo externo

O estilo externo tem esse título porquer ele é fora do html, isto é, é um arquivo de configuração de estilo assim sendo é independente das páginas e documentos;

Para aplicar estilo externo a tag é <Link>comando css;
Essa tag vai ficar dentro da tag <head> de preferência abaixo da tag <title> e todas as páginas vão ter essa tag de configuração externa de estilo não havendo necessidade de alterar estilo na página mas apenas copiando e colando a tag de configuração de estilo exteno permitindo personalizar todas as páginas de uma só vez;

Comando lembrete para criar estilo externo:
Ctrl+click seleção

Código modelo para arquivo css estilo externo:
h1Chaves aberta+Enter+Color e outros estilos+Chaves fechada;

É essencial não esquecer na primeira linha do arquivo css o comando de pontuação:
@charset "UTF-8";
sendo o @ para chamar o comando não esquecendo dos dois pontos no final;
